BoG Night

(Board Game Night) is a web service design to assist in the organisation and matchmaking of board game events. It supports regular events where members of the group bring their own games and is designed to make it easier to decide what to play first!

Board Game Geek The service can search the boardgamegeek.com database for details about individual games.

Beta It is currently in the Beta phase of development with many features to come, such as automated matchmaking based on individual user preferences and group game libraries so members can see what games might be available

Contact This project is under development by Jarryd van Zanten as part of a Business Technology degree project. Any comments and feedback are very greatly appreciated!
